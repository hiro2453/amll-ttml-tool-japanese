## 使用

> [!WARNING]
> 本リポジトリで公開しているものは Windows のみの実行となります。<br>オンライン版では日本語はないものの、MacユーザーやLinuxユーザーでもご利用できます。スマートフォンでのご利用はお勧めいたしません。

オンライン版は[こちら](https://steve-xmh.github.io/amll-ttml-tool/)から利用可能です

Github Action でビルドされた Tauri デスクトップ版も利用可能です，詳しくはこちらから → [Github Action 构建 Tauri 桌面版本](https://github.com/Steve-xmh/amll-ttml-tool/actions/workflows/build-test.yaml)。

## エディタ機能

- 基本入力・編集・軸入力機能
- TTML 形式の歌詞の読み込み・保存
- 歌詞の動作設定（バックボーカル、対唱歌詞など）
- 歌詞ファイルのメタデータ設定（タイトル，作者，Apple Music ID 等）
- 単語の分割・結合・移動
- LRC/ESLyric/YRC/QRC/Lyricify Syllable など歌詞ファイル形式のインポートや一部形式のエクスポートが可能
- 特殊記号付きプレーンテキスト歌詞のインポート対応
- カスタマイズ可能なショートカットキー

## 一部日本語対応

一部と言っても 10%ぐらい英語や中国語のままですが、使っていて支障がないぐらいには日本語に対応しています。
